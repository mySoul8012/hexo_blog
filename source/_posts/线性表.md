数据结构 - 线性表
 线性表就是一个队列
> 类似于node.js的队列，但是感觉很像，也不清楚是不是

直接上题，♪(^∇^*)

使用线性表来储存一组学生信息，并支持常规的增删查改
需要以下的几个子函数
1. 建立顺序表
2. 求线性表的长度
3. 输出线性表
4. 在线性表的指定位置插入一个元素
5. 根据键值查找指定的元素
6. 获取指定位置的元素信息
7. 删除指定位置的元素
8. 释放线性表
需要储存学生的信息有 
1. 学号
2. 姓名
3. 年龄
4. 专业
5. 入学年份

# 定义基本的数据类型
需要知道学生的学号，姓名，年龄，专业，入学年份，所以需要定义基本的数据类型
```
typedef struct {
    char num[20];
    char name[20];
    int age;
    char major;
    int registerYear;
}ElemType;
```
# 定义物理储存
定义线性表的物理储存方式
```
typedef struct {
    ElemType data[MAXCOUNT];    // 定义数据
    int len;    // 定义长度
};
```
以上完成了线性表的物理储存方式

// 移动线性表函数
// 其中option中1往后移动，0往前移动 col 为要插入的位置，从1开始
int moveList(SeqList *myList, int col, int option) {
    int i;
    // 判断传入的线性表是否为空
    if (myList == NULL) {
        printf ("线性表为空\n");
        return -1;
    }
    // 判断线性表是否已满
    if (listLength(myList) >= MAXCOUNT) {
        printf("线性表已满，无法插入，请删除后插入\n");
        return -1;
    }
    // 判断col是否越界，导致线性表不连贯
    if ((col > listLength(myList) && col - listLength(myList) != 1 ) || col < 1) {
        printf("输入的插入位置错误，请确认后重新插入\n");
        return -1;
    }
   // 判断前移还是后移
   switch(option) {
        case 1:
            // 线性表往后移动
           // 移动线性表
           col--;
           for(i = listLength(myList); i >= col; i--) {
               myList ->data[i] = myList ->data[i-1];
           };
           break;
        case 0:
            // 线性表往前移动
            for(i = col; i < listLength(myList); i++) {
                myList ->data[i - 1] = myList ->data[i];
            }
            break;
       default:
           printf("输入的选型错误\n");
           return -1;
    }
    return 1;
}
